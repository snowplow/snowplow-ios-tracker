Adds a dictionary of attributes to be appended into the SPPayload instance. Gives you the option to Base64 encode the data before adding it into the object. This method converts the dictionary to NSData and uses the data with addJsonStringToPayload:base64Encoded:typeWhenEncoded:typeWhenNotEncoded:

Parameters
jsonNSDictionary of JSON-compatible data to be added.
encodeBoolean option to choose whether the JSON data should be encoded.
typeEncodedIf the data is to be encoded, the result will be a value of the key in typeEncoded.
typeNotEncodedIf the data is NOT going to be encoded, the result will be a value of the key in typeWhenNotEncoded.
